To Clean/Compile:

1) Goto to unpacked folder (e.g. /home/xxxx/SherpaTTTeleClient/
	$> cd /home/xxxx/SherpaTTTeleClient

2) Clean last compilaton:
	$> ./clean.sh

3) Compile the sources
	$> ./build.sh

Generated API is deployed in /home/xxxx/SherpaTTTeleClient/build folder, being composed by:
	a) The SherpaTTTeleClient.h header file.
	b) The libSherpaTTTeleClient.so shared library.
